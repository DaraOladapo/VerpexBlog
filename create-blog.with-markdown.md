## Embedding code

I can even embed code with syntax highlighting:

JavaScript:

```js
const foo = 'bar';
```

Python:

```python
foo = 'bar'
```

Bash:

```bash
foo=bar
```

JSON:

```json
{
  "foo": "bar"
}
```

HTML:

```html
<div>foo</div>
```

CSS:

```css
.foo {
  color: red;
}
```

YAML:

```yaml
foo: bar
```

DIFF:

```diff
- foo
+ bar
```

GraphQL:

```graphql
{
  foo
}
```

Java:

```java
public class Foo {
  public static void main(String[] args) {
    System.out.println("foo");
  }
}
```

C#:

```csharp  
public class Foo {
  public static void Main(string[] args) {
    Console.WriteLine("foo");
  }
}
```
